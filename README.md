# Stock_market-Analysis
This project focuses on analyzing stock market data to identify trends, price movements, and key market signals. 
The analysis helps understand how different financial indicators—such as opening price, closing price, volume, volatility, change percentage, and market sentiment—affect stock performance over time.  

# Dependencies
**Prerequisites before running the project:**
- Windows 10 or above.
- Excel 2016 or later.
- Power BI Desktop (latest version).
- Keep the project folder structure unchanged.
- Ensure you have permission to access the data files.
  
 # Installation 
- Update file paths only if the data files were moved.
- Excel: Open Power Query → Check Source step.
- Power BI: Go to Edit Queries → Source → Update file path.
- Do not rename or remove folders unless documented.

 # Executing the Program
- Open the project folder and locate the main Excel (.xlsx) or Power BI (.pbix) file.
- Double-click the file to open it in Excel or Power BI Desktop.
- If prompted, click "Enable Editing" or "Enable Content".
- Excel: Go to Data → Refresh All.
- Power BI: Click Home → Refresh.
- If a data source error appears, update the file path in Power Query (Edit Queries → Source).
- Wait until the data is fully loaded with no warnings.
- Review the sheets/pages to confirm visuals and tables are updated.
- Save the file after successful execution.
  
# Help 
- If data fails to load → Check and update file path.
- If Excel/Power BI shows errors → Refresh again after path correction.
- If visuals appear blank → Ensure the dataset is not empty or filtered.
- If still unresolved → Restart Excel/Power BI and reopen the file.
